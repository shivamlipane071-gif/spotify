About This Project

This is a custom-built Spotify clone created by Shivam Lipane. It includes a playlist, play/pause controls, next/previous buttons, GIF animation, and volume control. The project is designed to be responsive and mobile-friendly.

The goal of this project is to practice full-stack development skills, experiment with HTML, CSS, and JavaScript, and create a personal portfolio project.
